# Blog

Personal blog built with [Hugo](https://gohugo.io/) and the [Blowfish](https://blowfish.page/) theme.

**Live site**: https://yagikota.github.io/blog/

## Local Development

```bash
# Install Hugo (macOS)
brew install hugo

# Start development server
hugo server -D
```

## Create New Post

```bash
hugo new posts/your-post-name/index.md
```

## Image Generate Prompt

```txt
Generate cyberpunk-style thumbnail images for each blog post in this folder.
Use a 16:9 aspect ratio and do not include any text or logos.
The style should feature neon lights, a dark futuristic city, high-tech and dystopian atmosphere, holograms, and a rainy night mood.
```
